# 🔐 Crypto Safety & Blockchain Learning Repository

A complete beginner-to-intermediate Python-based learning repository covering
cryptocurrency safety, trading strategies, security tools, and blockchain fundamentals.
Built for anyone who wants to understand and navigate the crypto world safely and confidently.

---

## 📁 Project Structure
```
crypto-safety/
│
├── crypto_safety_tips.py          # Beginner crypto safety tips
├── crypto_security_checker.py     # Advanced wallet security tools
├── crypto_trading_tips.py         # Trading strategies & risk tools
├── crypto_blockchain_basics.py    # Blockchain & crypto fundamentals
└── README.md                      # Project documentation
```

## 📦 Modules Overview

---

### 1. 🛡️ `crypto_safety_tips.py`
> **Level:** Beginner

A simple script that displays the 10 most essential cryptocurrency
safety tips every crypto user should know.

**What it covers:**
- Private key and seed phrase protection
- Two-factor authentication (2FA)
- Hardware wallet usage
- Phishing and scam awareness
- Safe exchange practices

**Run it:**
```bash
python crypto_safety_tips.py
```

---

### 2. 🔒 `crypto_security_checker.py`
> **Level:** Intermediate

An advanced security toolkit that validates wallet addresses,
checks password strength, hashes seed phrases, and generates
secure passwords.

- Bitcoin & Ethereum wallet address validator
- Password strength checker with scoring system
- SHA-256 seed phrase hashing simulation
- Cryptographically secure password generator

**Run it:**

## 🚀 Getting Started

### Prerequisites
- Python 3.6 or higher
- No external libraries required — uses Python standard library only

### Installation

1. Clone the repository:
```bash
   git clone https://github.com/Jeephoenix/crypto-safety.git
```

2. Navigate to the project folder:
```bash
   cd crypto-safety
```

3. Run any module:
```bash
   python crypto_safety_tips.py
   python crypto_security_checker.py
   python crypto_trading_tips.py
   python crypto_blockchain_basics.py
```
