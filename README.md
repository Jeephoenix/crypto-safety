# 🔐 Crypto Safety Tips

A simple Python script that displays essential cryptocurrency safety tips to help users protect their digital assets and avoid common crypto security mistakes.

---

## 📋 Description

This project provides a beginner-friendly list of the most important crypto safety practices. Whether you are new to cryptocurrency or an experienced trader, these tips serve as a quick reference guide to keep your assets safe.

---

## 🚀 Getting Started

### Prerequisites
- Python 3.6 or higher

### Installation

1. Clone the repository:
```bash
   git clone https://github.com/Jeephoenix/crypto-safety.git
```

2. Navigate to the project folder:
```bash
   cd crypto-safety
```

3. Run the script:
```bash
   python crypto_safety_tips.py
```

---

## 📌 What It Covers

| # | Tip |
|---|-----|
| 1 | Never share your private key or seed phrase |
| 2 | Always enable two-factor authentication (2FA) |
| 3 | Use a hardware wallet for long-term storage |
| 4 | Double-check wallet addresses before sending |
| 5 | Avoid public Wi-Fi when accessing your wallet |
| 6 | Keep your software and wallets updated |
| 7 | Use strong, unique passwords for each account |
| 8 | Beware of phishing links and fake websites |
| 9 | Only use reputable and well-known exchanges |
| 10 | Always backup your seed phrase offline |

---

## 💡 Sample Output
```
===== Crypto Safety Tips =====
1. Never share your private key or seed phrase with anyone.
2. Always enable two-factor authentication (2FA) on exchanges.
3. Use a hardware wallet for long-term storage.
4. Double-check wallet addresses before sending funds.
5. Avoid accessing your wallet on public Wi-Fi.
6. Keep your software and wallets updated regularly.
7. Use strong, unique passwords for each crypto account.
8. Beware of phishing links and fake websites.
9. Only use reputable and well-known exchanges.
10. Always backup your seed phrase offline in a safe place.
==============================
```

## 🛡️ Why Crypto Safety Matters

The crypto space is full of opportunities but also risks. Scams, hacks, and phishing attacks are common. A single mistake — like sharing your seed phrase or clicking a fake link — can result in permanent loss of funds. This script serves as a reminder of the basic habits every crypto user should practice daily.
